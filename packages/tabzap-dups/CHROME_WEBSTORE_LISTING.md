### Detailed Description (max 16,000 characters)
<!-- Full description shown on the extension's store page -->
TabZap Dups automatically detects and removes duplicate tabs to keep your browser organized and reduce memory usage.

**Features:**
• Automatic duplicate detection when new tabs open
• Manual scan with one click on the extension icon
• Configurable URL matching patterns
• Ignore list for sites you want to allow duplicates
• 5-second delay before closing (click icon to cancel)
• Works across all windows or current window only
• Pinned tabs are never closed

**How it works:**
When you open a tab that's already open elsewhere, TabZap Dups will automatically close the duplicate after a short delay. The badge shows how many duplicates are pending closure - click the icon to cancel.

Click the icon with no pending duplicates to scan and remove all duplicates at once.

**Privacy:**
This extension only accesses tab URLs to detect duplicates. No data is collected or transmitted.
