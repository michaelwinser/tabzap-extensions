### Detailed Description (max 16,000 characters)
<!-- Full description shown on the extension's store page -->
TabZap Expire automatically closes tabs that haven't been used in a while, helping you maintain a clean browser and reduce memory usage.

**Features:**
• Configurable inactivity timeout
• Whitelist domains you want to keep open
• Visual indicator of tab age
• Pinned tabs are never closed
• Active tab is never closed

**How it works:**
Tabs that haven't been viewed for longer than your configured timeout will be automatically closed. The extension tracks when you last viewed each tab and closes the oldest inactive ones.

**Use Cases:**
• Keep browser memory usage low
• Automatically clean up research tabs
• Maintain focus by reducing tab clutter
• Prevent "tab hoarding"

**Privacy:**
This extension only tracks tab activity timestamps locally. No data is collected or transmitted.
