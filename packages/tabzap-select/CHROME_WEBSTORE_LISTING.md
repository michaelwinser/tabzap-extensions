### Detailed Description (max 16,000 characters)
<!-- Full description shown on the extension's store page -->
TabZap Select lets you quickly select multiple tabs and perform batch operations on them. Stop closing tabs one by one!

**Features:**
• Select multiple tabs with checkboxes
• Close all selected tabs at once
• Bookmark selected tabs to a folder
• Copy URLs of selected tabs
• Filter tabs by title or URL
• Select all / deselect all

**How it works:**
Click the extension icon to open the tab selector popup. Check the tabs you want to act on, then choose an action. It's that simple!

**Batch Operations:**
• **Close** - Close all selected tabs
• **Bookmark** - Save selected tabs to a bookmark folder
• **Copy URLs** - Copy URLs to clipboard (one per line)

**Use Cases:**
• Clean up after a research session
• Save a group of related tabs as bookmarks
• Share multiple URLs at once
• Quickly close all tabs from a specific site

**Privacy:**
This extension only reads tab information to display the selector. No data is collected or transmitted.
